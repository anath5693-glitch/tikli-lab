# PinRup AI Lab – Netlify Redeploy Package

This package includes the updated static website content for PinRup AI Lab.

## Pages
- index.html
- services.html
- about.html
- blogs.html
- contact.html
- faq.html
- legal/privacy-policy.html
- legal/terms.html
- legal/cookies.html

## Redeploy on Netlify
1. Download and unzip this package.
2. Go to your Netlify site dashboard.
3. Open the Deploys tab.
4. Drag and drop the unzipped folder contents or re-upload the package as a new deploy.

## Before launch
- Replace legal placeholder content.
- Connect the form to Netlify Forms or another form handler.
- Update metadata/domain links if needed.
